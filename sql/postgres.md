### PostgreSQL

```
sudo apt update
```
```
sudo apt install postgresql
```
* БД создается с дефолтным пользователем postgres
* запускаем консоль от имени пользователя postgres
```
sudo -u postgres psql
```
* посмотреть все БД
```
\l
```
* посмотреть всех пользователей
```
\du
```
* создать БД
```
CREATE DATABASE db_name;
```
* зайти под БД
```
\c db_name
```
* список таблиц БД
```
\d
```
* посмотреть таблицу
```
\d table_name
```
